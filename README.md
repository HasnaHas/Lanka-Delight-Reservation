# Lanka Delight

A full-stack restaurant reservation and menu management system for **Lanka Delight**.

---

## Features

### User Side
- Browse and filter menu items by category (Lunch, Dinner, Side, Other) without logging in.  
- Fill in a reservation form easily without creating an account.  

### Admin Side
- Manage menu items: add, edit, delete, and upload images.  
- View all reservations submitted by users.  
- Manually review reservations and update their status as **Accepted** or **Declined**.  
- Send email notifications to users after reviewing reservations.  

---
## Tech Stack

- **Frontend:** React, Tailwind CSS  
- **Backend:** Node.js, Express.js, MongoDB  
- **Hosting:** Vercel (Frontend)  for demo purpose
